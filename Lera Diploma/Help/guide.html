ИС финансового отдела — краткое руководство

Навигация по разделам — слева. В шапке каждого экрана есть кнопка Справка и горячая клавиша F11 (открывает HTML-справку по текущему модулю).

Главное меню

Выберите период (или «Произвольно» и даты «С»/«По»), нажмите Обновить. Карточки KPI показывают количество проведённых и черновых документов, оборот по проводкам и число контрагентов в периоде.

Графики: линия — суммы проводок по дням (только проведённые документы); столбцы — суммы по типам документов. Подписи осей: дата / тип и сумма в рублях. Наведите курсор ближе к точке или столбцу — всплывающая подсказка с датой (или типом) и суммой.

Внизу — последние документы; щелчок по заголовку столбца сортирует таблицу.

Документы

Поиск по номеру и тексту, фильтр по статусу и датам. Текст поиска подсвечивается в найденных ячейках. Сортировка — щелчок по заголовку столбца. Один щелчок по строке выделяет её; двойной щелчок открывает редактирование. Проведение — кнопка «Провести» при наличии права.

Журнал проводок

Аналогично документам: поиск с подсветкой, фильтры, сортировка по столбцам. Двойной щелчок открывает документ, к которому относится проводка.

Контрагенты

Справочник контрагентов. Двойной щелчок по строке открывает карточку (при праве на редактирование). ИНН/КПП проверяются при сохранении.

Статьи бюджета

Код и наименование статей для привязки расходов в документах.

Справочники

Две области: типы документов и статусы. Разделитель можно двигать. Редактирование — по правам роли.

План счетов

Иерархия счетов; при необходимости укажите родительский счёт.

Отчёты

Выберите вид отчёта, период и при необходимости счёт. После «Сформировать» доступны таблица, диаграммы и экспорт (по правам).

Пользователи

Управление учётными записями. Колонка Активен: щёлкните по галочке, чтобы включить или выключить пользователя (нельзя отключить самого себя). Кнопка «Вкл./выкл.» дублирует действие для выбранной строки.

Журнал аудита

Просмотр действий пользователей; сортировка по столбцам.

Резервное копирование

Важно: SQL Server записывает .bak от имени службы SQL Server, а не вашего пользователя Windows. Путь на рабочий стол часто недоступен службе. Укажите локальный путь на сервере, например C:\SQLBackup\Finance.bak, предварительно выдав права на папку учётной записи SQL Server.

Вход и общие сведения

Тестовые учётные записи см. в общей справке. После входа используйте боковое меню. При ошибках обращайте внимание на текст сообщения — к резервным копиям и правам часто добавляются пояснения.
