# ИС финансового отдела (Дипломный проект)

Настольное приложение для учёта финансовых документов, проводок, справочников, отчётов и администрирования пользователей. Интерфейс на русском языке.

## Технологии

- **.NET Framework 4.8**, **Windows Forms**
- **Entity Framework 6** (Code First, инициализация и сиды при первом создании БД)
- **SQL Server** — строка подключения `FinanceContext` в `App.config`
- Дополнительно: **ClosedXML** (Excel), **PDFsharp** (PDF), **BCrypt** (хэши паролей), **Chart** (дашборд и отчёты)

## Требования

- Windows с установленным **.NET Framework 4.8** (или SDK для сборки)
- Доступный экземпляр **Microsoft SQL Server** (или LocalDB), права на создание/использование базы согласно настройкам приложения

## Запуск и настройка

1. В файле **`App.config`** укажите актуальную строку подключения в элементе `connectionStrings` → имя **`FinanceContext`** (сервер, база, проверка подлинности).
2. Соберите решение и запустите `Lera Diploma.exe` из папки `bin\Debug\net48` или `bin\Release\net48`.
3. При первом запуске выполняется инициализация БД (`DatabaseBootstrapper`).

Сборка из каталога проекта:

```bash
dotnet build "Lera Diploma.csproj"
```

## Вход (после первого сида БД)

| Логин   | Пароль (демо) | Роль              |
|---------|----------------|-------------------|
| `admin` | `admin`        | Администратор     |
| `gbuh`  | `demo`         | Главный бухгалтер |

Пароли и пользователи задаются в `FinancialDbInitializer` при пустой базе; при необходимости измените сиды до первого деплоя.

## Основные возможности

- Главное меню (KPI, графики, последние документы)
- Документы и журнал проводок, контрагенты, статьи бюджета, справочники, план счетов
- Отчёты с выгрузкой в Excel/PDF и печатью
- Пользователи, роли и права, журнал аудита
- Резервное копирование и восстановление БД (`.bak`; путь должен быть доступен **учётной записи службы SQL Server**)

## Структура каталогов (кратко)

| Папка        | Назначение                          |
|-------------|--------------------------------------|
| `Controls`  | Экранные модули (UserControl)        |
| `Forms`     | Главная форма, вход, модальные окна |
| `Data`      | DbContext, инициализация БД         |
| `Models`    | Сущности EF                          |
| `Services`  | Бизнес-логика и отчёты               |
| `Security`  | Контекст пользователя, права         |
| `UI`        | Тема, тулбары, экспорт, подсказки    |
| `Help`      | Справка (копируется в выходной каталог) |
| `Database`  | `Init.sql` и вспомогательные скрипты |

## Справка в приложении

Клавиша **F11** открывает контекстную справку по текущему модулю (если для него есть раздел в `Help`).
